# Landing Page — Como Anunciar no Instagram de Forma Profissional

Landing page estática (HTML/CSS/JS puro, sem build) para o produto digital
"Como Anunciar no Instagram de Forma Profissional", vendido na Hotmart.

- Domínio: anuncioprofissional.com.br
- Checkout: https://pay.hotmart.com/M106420803Y
- Hospedagem: Vercel (deploy estático, sem configuração adicional necessária)

## Estrutura

Arquivo único `index.html` com CSS embutido (`<style>`) e JS vanilla
(`<script>`) para: scroll reveal, accordion do FAQ, contador regressivo de
urgência e botão flutuante do WhatsApp.

## Deploy na Vercel

1. Importe este repositório em vercel.com/new
2. Framework preset: **Other** (site estático — nenhuma variável de ambiente é necessária)
3. Após o deploy, configure o domínio `anuncioprofissional.com.br` em
   Project Settings → Domains
